# BioVisionTracker
A MATLAB-based software for the semi-automated analysis of Arabidopsis thaliana fluorescence microscopy images.

Installation instructions
1. Download all contents of the BVT software into a single folder on your local computer.
2. Open MATLAB.
3. Click "Home>Set Path" on the top bar of the window. This will cause a "Set Path" window to appear.
4. Click the "Add With Subfolders" option.
5. Navigate to and click the folder that contains the BVT software and click "Select Folder."
6. Click "Save" in the "Set Path" window and close the "Set Path" window. 
7. The BVT software is now installed and ready to use.
